## Run Locally

To run the project locally, follow the steps below:

Clone the project

```bash
  git clone https://github.com/briannval/StarryPort.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
